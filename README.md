# 🏠 Dotfiles ✨

My personal macOS developer terminal configuration.

## 🧰 What's included

- 👻 **Ghostty** — terminal with purple theme
- 🐟 **Fish shell** — with cloud CLI shortcuts and abbreviations
- 🚀 **Starship** — prompt with git, AWS, GCP, Azure, Kubernetes
- 💎 **Neovim 0.11** — full IDE setup with LSP, treesitter, telescope
- 📂 **eza** — modern ls replacement

See [SETUP.md](SETUP.md) for step-by-step installation instructions.

## 📋 Requirements

- 🍎 macOS
- 🍺 Homebrew
- 🔤 JetBrainsMono Nerd Font
- 🟩 Node.js (via nvm)
- 🐹 Go
